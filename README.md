### :electric_plug: Instructions to run this project
1. On terminal type: 
  
    ``` git clone {the url to the GitHub repo} ```
 
2. ``` cd ``` into the folder and type 
 
    ``` $ npm install ```
3. To run the project 

    ```$ npm start```
